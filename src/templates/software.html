2026常用科学上网客户端下载与配置指南

为了方便大家快速配置订阅出海，我们整理了 Windows、macOS、Android、iOS 各大平台主流代理软件的官方下载链接及图文使用教程。请根据您的系统选择合适的软件。

Windows macOS 首选推荐

Clash Verge Rev

基于 Tauri 架构开发的全新一代 Clash 桌面客户端。内置 Mihomo (Meta) 核心，全面兼容旧版 Clash Verge，内存占用极小，完美支持 VLESS、Reality、Hysteria2 等最新加密协议，界面高雅精美，原生支持中文。

⚙️ 快速配置步骤：
  1. 从机场后台复制 Clash 订阅链接。
  2. 打开软件，点击左侧的 Profiles 选项卡。
  3. 在顶部输入框粘贴订阅链接，点击右侧的 Import。
  4. 下载完成后，双击激活该订阅文件（显示绿色）。
  5. 进入 Proxies 页面选择节点，并在 Settings 开启 System Proxy 系统代理开关。

iOS / iPadOS iOS必备

Shadowrocket (小火箭)

iOS 平台最经典的“国民级”规则代理工具。性能强劲稳定，协议支持极度全面，能无缝解析并自动导入各大机场的订阅链接。操作界面极其傻瓜化，适合绝大多数苹果用户。

⚙️ 快速配置步骤：
  1. 由于政策原因，需登录非中国大陆区的 Apple ID 并在 App Store 购买下载小火箭。
  2. 登录机场后台，复制订阅链接。
  3. 打开 Shadowrocket，点击右上角 ＋，类型选择 Subscribe。
  4. 在 URL 中粘贴订阅链接并保存，回到主页即可看到节点列表。
  5. 开启顶部“未连接”开关，同意安装 VPN 描述文件，即可开启代理。

Windows 老牌经典

v2rayN (Windows)

Windows 系统下最为经典、低调且强大的核心代理客户端。对新底层协议适配速度极快，无内存泄漏风险，能够承载海量并发连接，非常适合极客玩家和需要长时间稳定工作的商务用户。

⚙️ 快速配置步骤：
  1. 复制机场的 v2ray 订阅地址。
  2. 打开 v2rayN，点击顶部菜单的 订阅分组 -> 订阅分组设置，点击“添加”。
  3. 填写别名，并在 可选地址(url) 中粘贴订阅链接，点击“确定”。
  4. 返回主页，点击 订阅分组 -> 更新全部订阅(不通过代理) 即可加载节点。
  5. 右键右下角托盘图标，将 系统代理 设为“自动配置系统代理”，路由模式推荐“绕过大陆”。

Android 安卓首选

Clash Meta (Android)

安卓端（目前已更名为 Mihomo for Android）功能最强大、对新式协议支持最完美的科学上网神器。内置高性能 Meta 内核，相比旧版 Clash for Android 具有更优越的防断连性能与路由分流策略。

⚙️ 快速配置步骤：
  1. 在手机浏览器中登录机场后台，复制 Clash 订阅链接。
  2. 打开 Clash Meta 客户端，进入 配置（Profiles） 菜单。
  3. 点击右上角 ＋，选择 URL 导入。
  4. 输入备注并粘贴链接，点击保存后等待订阅文件配置下载完成。
  5. 选中该配置，返回主界面点击 启动（Start） 并在代理中选择节点即可。

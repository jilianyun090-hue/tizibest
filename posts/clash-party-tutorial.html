一、什么是 Clash Party？

Clash Party 是基于开源 Clash 项目深度定制的桌面端分支版。在瞬息万变的网络环境中，它坚持"稳定大于一切"的设计理念，特别适合追求极致稳定性的用户。

二、🚀 软件亮点

  • 极致稳定：以稳定性为第一设计目标，适合长期挂机使用。
  • 兼容性强：支持主流机场的所有订阅格式。
  • 规则分流：内置强大的规则分流引擎，国内直连国外走代理。

三、📥 软件下载

官方 GitHub 下载地址：Clash Party (GUI for Clash) Releases

根据系统选择对应的安装包下载：

  • Windows：下载 .exe 安装包，双击安装即可。
  • macOS：下载 .dmg 文件，拖入 Applications 文件夹。

⬇️ GitHub 官方下载 Clash Party

四、⚙️ 添加订阅教程

导入订阅链接

打开 Clash Party，点击 配置 → 导入订阅，粘贴机场提供的订阅链接，点击确认。

选择节点开启代理

订阅更新完成后，在节点列表里选择延迟较低的节点，然后打开主界面的代理开关。

切换代理模式

点击模式按钮，可以在以下模式之间切换：

  • 规则（推荐）：国内直连，国外走代理，节省流量。
  • 全局：所有流量均走代理，适合访问全部海外服务。
  • 直连：相当于关闭代理，用于网络排障。

五、💡 使用建议

📌 日常使用：规则模式（国内直连，国外走代理）

🚀 全局代理：访问全部海外服务时使用

🔄 节点不通：切换节点或更新订阅

⚡ 速度慢：优先选择低延迟节点（SG / JP / HK）

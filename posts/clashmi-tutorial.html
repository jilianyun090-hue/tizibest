一、什么是 Clash Mi？

Clash Mi 是一款基于 Mihomo（Clash Meta）核心 的代理客户端，界面简洁、操作直观，非常适合新手快速导入机场订阅并使用规则分流或全局代理。

二、🚀 软件亮点

  • 极致精简：无多余动画与广告，一切为了极致的网络转发效率。
  • 内核无缝适配：完美发挥 Mihomo 内核在流媒体解锁与边缘分流上的优势。
  • 资源占用低：内存占用仅为同类 Electron 框架软件的不到三分之一。
  • 配置灵活：支持复杂的 YAML 自定义，满足各种专家级分流场景。

三、📥 软件获取

官方 GitHub 下载地址：Clash Mi Releases

建议优先下载最新正式版（Release），根据系统选择对应安装包。

⬇️ GitHub 官方下载 Clash Mi

四、⚙️ 添加订阅教程

新增配置

打开 Clash Mi：

  • 进入首页
  • 点击 我的配置
  • 点击右上角 ＋

选择「从剪贴板导入」

提前复制好你的机场订阅链接，然后选择：从剪贴板导入

粘贴订阅链接并填写备注

在配置页面中：

  • 配置链接 / 内容：粘贴订阅地址
  • 备注：随意填写（如「我的订阅」）
  • 点击右上角 ✔ 保存

导入完成

看到提示后，点击 确定 即表示订阅导入成功。

五、🚀 启动代理连接

返回主界面

点击左上角返回首页。

开启连接

在首页进行以下操作：

  • 代理模式：建议选择「规则」或「全局」
  • 打开右上角开关
  • 状态显示 已连接 即表示代理生效

六、🔄 节点切换方法

如果你的订阅包含多个节点：

  1. 点击首页 代理
  2. 进入节点列表
  3. 选择需要使用的节点（如 🇭🇰 香港 / 🇸🇬 新加坡 / 🇺🇸 美国）

💡 使用建议：日常使用推荐规则模式，国内直连节省流量；如果遇到访问不了的特殊场景，再切换为全局模式。节点速度慢时优先选择低延迟节点（SG / JP / HK）。
